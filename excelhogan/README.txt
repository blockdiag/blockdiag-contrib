===========================
blockdiagcontrib-excelhogan
===========================
A plugin for `blockdiag` that provides imagedrawer for excel hogan.
Many Japanese peaple love writing documentation with MS-Excel.


Usage
======
Give '-T hogan.xlsx' option to blockdiag command line::

   $ blockdiag -T hogan.xlsx input.diag


Requirements
============
* blockdiag 1.2.0 or later
