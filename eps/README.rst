=====================
blockdiagcontrib-eps
=====================
A plugin for `blockdiag` that provides imagedrawer for EPS (Encapsulated PostScript).


Usage
======
Give '-T eps' option to blockdiag command line::

   $ blockdiag -T eps input.diag


Requirements
=============
* blockdiag 1.4.0 or later
