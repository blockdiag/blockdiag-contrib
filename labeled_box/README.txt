A plugin for `blockdiag` that provides labeled_box node-renderer

`blockdiagcontrib-labeled_box` is a sample implementation for blockdiag plugin.

Diagram examples
================

`blockdiagcontrib-labeled_box` renders class as node's shape.

Example::

    diagram admin {
      A [shape = labeled_box, toplabel = "label"];
    }


Requirements
============

* blockdiag 1.1.8 or later
