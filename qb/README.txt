A plugin for `blockdiag` that provides QB node-renderer

`blockdiagcontrib-qb` is a sample implementation for blockdiag plugin.

Diagram examples
================

`blockdiagcontrib-qb` renders QB as node's shape.

Example::

    diagram admin {
      A [shape = qb];
    }


Requirements
============

* blockdiag 0.7.0 or later
