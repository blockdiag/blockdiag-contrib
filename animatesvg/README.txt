============================
blockdiagcontrib-animatedsvg
============================
A plugin for `blockdiag` that provides imagedrawer for animated svg.


Usage
======
Give '-T animated.svg' option to blockdiag command line::

   $ blockdiag -T animated.svg input.diag


Requirements
============
* blockdiag 1.2.4 or later
