A plugin for `blockdiag` that provides Square node-renderer

`blockdiagcontrib-square` is a sample implementation for blockdiag plugin.

Diagram examples
================

`blockdiagcontrib-square` renders square as node's shape.

Example::

    diagram admin {
      A [shape = square];
    }


Requirements
============

* blockdiag 0.7.0 or later
